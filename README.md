## 💡 주요 기능

- Main Page : 홈 화면, 상품 카테고리별 서브 메뉴, 사이드 바, 상품 미리보기, 이벤트 및 신제품 홍보용 배너
- Item List Page : 상품 목록, 서브카테고리에 따른 상품 종류 선택, 필터숨기기, 정렬기준: [추천순, 최신순, 높은가격순, 낮은가격순], 가격대 필터링
- Item Detail Page : 색상 선택, 사이즈 선택, 장바구니에 해당 상품 담기, 제품 추천
- Cart Page : 장바구니에 담긴 제품 목록 표시, 수량 조절/삭제, 제품 추천
- Sign In / Sign Up Page : 이메일/비밀번호로 로그인(이메일이 가입되어 있을 시), 회원가입(이메일이 가입되어 있지 않을 시)
- Map Store Page : 나이키 매장 리스트 지도에 표시
